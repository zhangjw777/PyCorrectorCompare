# 项目修改计划

## 项目目标
使用 pycorrector 库中不同的纠错模型对错误句子数据集进行检错，评估各模型的检错能力，与自研模型进行对比。

## 评估指标
- Precision（精确率）
- Recall（召回率）
- F0.5
- F1
- F2
